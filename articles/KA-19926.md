# AEP - Se un set di dati non è abilitato per Profilo, in che modo influisce sulla segmentazione?

## Descrizione

Se un set di dati non è abilitato per Profilo, i dati inclusi in tale set di dati non vengono considerati per la qualificazione della segmentazione (anche se nel set di dati/schema è presente un’identità specifica)?

## Risoluzione


No, non è affatto considerato, è così che funziona l&#39;Experience Platform. La segmentazione funziona a livello di UPS (Unified Profile Service) e non a livello di Data Lake.

Se il profilo non è abilitato, i dati che entrano nel Data Lake non vengono promossi ai servizi Unified Profile e Identity
