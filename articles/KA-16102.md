# Eccezione NullPointer durante il recupero dell’elenco di configurazione di destinazione


Questo articolo aiuta a risolvere i *Eccezione NullPointerException* errore che si verifica durante il recupero dei file di configurazione di target dopo l’aggiornamento a Adobe Experience Manager 6.5.

## Descrizione {#description}


### <b>Ambiente</b>

Adobe Experience Manager 6.5

## Risoluzione {#resolution}


È necessario verificare se nel sistema è presente una configurazione di destinazione non valida

In caso affermativo, dobbiamo rimuoverlo.

Il riferimento della presente risoluzione è tratto dal codice implementato in - `/libs/cq/personalization/touch-ui/components/datasource/targetconfigdatasource/targetconfigdatasource.jsp`
