# Errore durante l’importazione di html in E-mail Designer

## Descrizione

<b>Problema:</b><br><br> <br><br>Durante l’importazione di un file .zip con un file HTML e immagini in E-mail Designer, si è verificato l’errore seguente:<br><br> <br><br>*(1:117056) : Riferimento carattere non valido

## Risoluzione


<b>Motivo:</b>
