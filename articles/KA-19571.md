# Come controllare i nodi aggiornati da un&#39;installazione del service pack

## Descrizione


Quando installi un service pack come AEM 6.5.12.0, potresti voler sapere quali nodi vengono aggiornati. Questo articolo spiega come controllare questi nodi con un log di debug.

<b>Ambiente</b>
AEM 6.5


## Risoluzione


Prima di installare il service pack, configura un registro di debug per org.apache.jackrabbit.vault.packaging.impl.ActivityLog. Il registro di debug genera i percorsi dei nodi aggiornati.

1. Vai al supporto per il registro Sling della console Web (http://localhost:4502/system/console/slinglog)
2. Pulsante Seleziona &quot;Aggiungi nuovo logger&quot;
3. Compila i seguenti valori, quindi seleziona il pulsante &quot;salva&quot;

Esempio di configurazione:    - loglevel = DEBUG - file di registro = logs/activity.log - logger = org.apache.jackrabbit.vault.packaging.impl.ActivityLog

4. Installare il service pack
5. Controlla il registro di debug
