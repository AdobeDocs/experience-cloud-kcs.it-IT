<b>Causa</b>
Alcuni file JAR mancanti sono segnalati dal processo WEB.


## Risoluzione


<b>Soluzione</b>

La `classpath` dal tag `serverConf.xml` il file di configurazione non è stato aggiornato correttamente dopo l&#39;aggiornamento.

La `classpath` Il tag deve puntare alle JAR più recenti.
